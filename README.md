# Repositório sobre nodemcu - ESP-8266

## Imagem

Existe uma imagem já compilada disponível para download dentro deste repositório. Basta ir na pasta [image](./image) e utilizar o software de atualização de imagem para gravá-lo. 

## QuickStart

Para detalhes completos, acesse o post [Hello world em lua com nodemcu](https://programacaoeletronica.wordpress.com/2022/04/11/hello-world-em-lua-com-nodemcu/).

### Software de atualização de imagem

Os tutoriais do blog são feitos utilizando o software [nodemcu flasher](https://github.com/nodemcu/nodemcu-flasher).

### Software de programação e atualização de arquivos .lua

Para a programação, indico o software [ESPlorer](https://github.com/4refr0nt/ESPlorer/releases).
